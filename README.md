# 🚀 QuickOrder AI

### Intelligent Food Ordering & Tracking Chatbot

QuickOrder AI is an end-to-end food ordering chatbot that enables users to place orders, modify existing orders, and track order status through natural language conversations. The chatbot leverages Dialogflow ES for intent recognition, FastAPI for webhook handling, and MySQL for order management and tracking.

---

## ✨ Features

* 🍕 Place food orders through conversational interactions
* 📝 Add or remove food items from an order
* 🔢 Automatically generate unique Order IDs
* 📦 Track order status in real time
* 💾 Store and manage orders using MySQL
* 🤖 Natural language understanding powered by Dialogflow ES

## 🚀 How to Run

### 1. Start FastAPI Server

```bash
python -m uvicorn main:app --reload
```

### 2. Expose the Server Using ngrok

```bash
ngrok http 8000
```

### 3. Update Dialogflow Webhook URL

Copy the HTTPS URL generated by ngrok and configure it in Dialogflow Fulfillment settings.

### 4. Test the Chatbot

Use the Dialogflow simulator to:

* Place orders
* Remove items
* Complete orders
* Track existing orders
