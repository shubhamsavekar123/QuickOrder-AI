# 🚀 QuickOrder AI

### Intelligent Food Ordering & Tracking Chatbot

QuickOrder AI is a food ordering chatbot that enables users to place orders, modify existing orders, and track order status through natural language conversations. The project leverages Dialogflow ES for intent recognition, FastAPI for webhook handling, MySQL for order management, and Streamlit for the frontend interface.

---

## 📌 Project Status

🚧 **Currently Under Development**

The core functionalities, backend logic, database integration, and Streamlit frontend have been implemented successfully. I am currently working on resolving Dialogflow webhook connectivity and authentication-related issues to achieve complete end-to-end integration.

This project serves as a hands-on learning experience in Conversational AI, NLP, backend development, and database integration.

---

## ✨ Features

* 🍕 Place food orders through conversational interactions
* 📝 Add or remove food items from an order
* 🔢 Automatically generate unique Order IDs
* 📦 Track order status
* 💾 Store and manage orders using MySQL
* 🤖 Intent recognition powered by Dialogflow ES
* 🖥️ Streamlit-based frontend prototype

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/QuickOrder-AI.git
cd QuickOrder-AI
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Start FastAPI Server

```bash
python -m uvicorn main:app --reload
```

### 4. Expose the Server Using ngrok

```bash
ngrok http 8000
```

### 5. Configure Dialogflow Webhook

Copy the HTTPS URL generated by ngrok and update the webhook URL in Dialogflow Fulfillment settings.

### 6. Test the Chatbot

Use the Dialogflow simulator to:

* Place orders
* Add items
* Remove items
* Complete orders
* Track order status
